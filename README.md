# Crop Disease Detection - MLOps on Kubernetes

## Project Title
A Kubernetes-Based MLOps Framework for Scalable Deep 
Learning-Driven Crop Disease Detection in Smart Agriculture

## Domain
Artificial Intelligence | Smart Agriculture | MLOps | Cloud Computing

## College
Atria Institute of Technology, Bengaluru
Department of Information Science & Engineering - 6ISE

## Objectives
1. Detect crop diseases from leaf images using Deep Learning
2. Build automated MLOps pipeline for training and deployment
3. Containerize the system using Docker
4. Scale using Kubernetes cluster
5. Expose REST API for real-time prediction
6. Monitor performance using Prometheus and Grafana

## Tech Stack
- Python, TensorFlow, EfficientNet
- MLflow, Docker, Kubernetes
- FastAPI, Prometheus, Grafana
- GitHub Actions (CI/CD)

## Dataset
PlantVillage Dataset - 54,000+ images, 38 disease classes
