About
About this project

This is a final year B.E project from the Department of Information Science and Engineering at Atria Institute of Technology, Bengaluru.

We built a complete MLOps pipeline using Kubernetes for scalable deep learning-driven crop disease detection in smart agriculture.

The system uses EfficientNetB0 transfer learning trained on the PlantVillage dataset with 54,000+ images across 38 disease classes.

Python TensorFlow EfficientNet FastAPI Docker Kubernetes MLflow Prometheus Grafana
